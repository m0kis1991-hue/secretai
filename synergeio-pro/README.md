# ΣυνεργείοPro — Διαχείριση Service για Συνεργεία

PWA (Progressive Web App) για διαχείριση πελατών, οχημάτων (αυτοκινήτων / μηχανών / σκαφών / φορτηγών) και ιστορικού service. Δουλεύει σε browser αλλά εγκαθίσταται και ως εφαρμογή σε κινητό/desktop με offline λειτουργία.

## Τι περιλαμβάνει

- **Διαχείριση πελατών** — ονοματεπώνυμο, τηλέφωνο, email, ΑΦΜ, διεύθυνση, σημειώσεις. Cascade διαγραφή (όχημα + service).
- **Διαχείριση οχημάτων** — 4 τύποι (αυτοκίνητο, μοτοσικλέτα, σκάφος, φορτηγό) με μάρκα, μοντέλο, χρονολογία, πινακίδα, VIN, κυβισμό, κωδικό κινητήρα, καύσιμο, χρώμα, χιλιόμετρα, φωτογραφία άδειας.
- **Καταγραφή service** — τύπος (αλλαγή λαδιών, φρένα, μεγάλο service κ.λπ.), περιγραφή εργασιών, λίστα ανταλλακτικών με κωδικό/ποσότητα/τιμή, ώρες εργασίας × τιμή ώρας, υπολογισμός συνόλου, χιλιόμετρα τότε, μηχανικός, επόμενο service.
- **Έξυπνες υπενθυμίσεις** — αυτόματος υπολογισμός επόμενου service (βάσει διαστήματος μηνών/χλμ). Dashboard με ληγμένα και επερχόμενα. Αποστολή υπενθύμισης σε **WhatsApp**, **Viber**, **SMS** ή **email** με ένα κλικ, με προσυμπληρωμένο μήνυμα (επεξεργάσιμο πριν την αποστολή).
- **AI σκανάρισμα άδειας κυκλοφορίας** — τραβάς φωτό και το AI (OpenAI GPT-4o vision) εξάγει αυτόματα πινακίδα, VIN, μάρκα, μοντέλο, χρονολογία, κυβισμό, καύσιμο, χρώμα. Λειτουργεί σε DEMO mode χωρίς κλειδί (mock δεδομένα) ή πραγματικά με το API key σου.
- **Έξυπνη αναζήτηση ανταλλακτικών** — deep links σε Skroutz, Car-Parts.gr, Autodoc, Oscaro, eBay Motors, Google με VIN/μάρκα/μοντέλο/έτος. Αναζήτηση και με ελεύθερο κείμενο.
- **Εξαγωγή PDF** — επαγγελματικό report του service με στοιχεία συνεργείου/πελάτη/οχήματος/εργασιών/ανταλλακτικών/συνόλων.
- **Κοινοποίηση στον πελάτη** — σύνοψη service ή ολόκληρου ιστορικού σε WhatsApp/Viber/email/clipboard.
- **Dashboard** — αριθμός πελατών/οχημάτων, service του μήνα, έσοδα μήνα, γρήγορες ενέργειες, επόμενες υπενθυμίσεις, πρόσφατα service.
- **Backup/Restore** — εξαγωγή/εισαγωγή όλων των δεδομένων ως JSON.
- **Bilingual UI** — Ελληνικά + English (αλλαγή live από Ρυθμίσεις).
- **Theme** — φωτεινό / σκοτεινό / αυτόματο.
- **PWA** — εγκαθίσταται από browser σε κινητό/desktop, δουλεύει offline μετά το πρώτο load.

## Πρόσθετα χρήσιμα features που πρόσθεσα

- **Αυτόματη ενημέρωση χιλιομέτρων**: όταν κάνεις νέο service, αν τα χιλιόμετρα είναι μεγαλύτερα από αυτά του οχήματος, ενημερώνεται αυτόματα.
- **Σύνδεσμοι Web Share API**: σε υποστηριζόμενες συσκευές κινητού, χρήση native share dialog για ταχύτερη αποστολή.
- **Έξυπνος καθαρισμός τηλεφώνου**: τα ελληνικά κινητά παίρνουν αυτόματα πρόθεμα +30 ώστε να δουλεύουν σωστά τα WhatsApp/Viber links.
- **Επανασύνδεση οχημάτων με VIN**: το AI scan αναγνωρίζει υπάρχοντα οχήματα από VIN.
- **Search & filter** σε όλες τις λίστες.
- **Cascade διαγραφή**: όταν διαγράφεις πελάτη, διαγράφονται και τα οχήματα + service του.

## Πώς το τρέχεις τοπικά

### Επιλογή 1 — Με Python (συνιστάται)

```bash
python serve.py
```

Άνοιξε το `http://localhost:8000` σε browser (Chrome, Edge, Safari, Firefox).

### Επιλογή 2 — Με Node

```bash
npx http-server -p 8000
```

### Επιλογή 3 — Με VS Code

Εγκατέστησε το extension **Live Server** και πάτα "Go Live".

> ⚠️ Άνοιγμα του `index.html` με διπλό κλικ (`file://`) **δεν** ενεργοποιεί το service worker (δεν θα δουλέψει η εγκατάσταση/offline). Το υπόλοιπο της εφαρμογής δουλεύει κανονικά.

## Πώς εγκαθίσταται σαν app

### Android (Chrome / Edge / Brave)
1. Άνοιξε το URL που τρέχει η εφαρμογή.
2. Μενού → "**Εγκατάσταση εφαρμογής**" ή "**Add to Home screen**".

### iOS (Safari)
1. Άνοιξε σε Safari (όχι Chrome).
2. Κουμπί Share → "**Προσθήκη στην Αφετηρία**".

### Desktop (Chrome / Edge)
- Δες το εικονίδιο εγκατάστασης στη γραμμή διευθύνσεων δεξιά.

## Σύνδεση πραγματικού AI για σκανάρισμα άδειας

1. Βγάλε API key από [OpenAI](https://platform.openai.com/api-keys).
2. Πήγαινε Ρυθμίσεις → AI Settings → επικόλληση κλειδιού.
3. Αποθηκεύεται μόνο τοπικά στη συσκευή (localStorage). Δεν φεύγει πουθενά.

> Χωρίς API key η εφαρμογή τρέχει σε **DEMO mode** με προσομοιωμένα δεδομένα — μπορείς να δοκιμάσεις την ροή χωρίς κόστος.

Το πρότυπο prompt μπορεί να αλλαχθεί στο `utils.js` (συνάρτηση `aiExtractRegistration`). Μπορείς να βάλεις και Anthropic Claude — απλά άλλαξε το endpoint και τη δομή.

## Δομή φακέλων

```
synergeio-pro/
├── index.html              ← shell, layout, sidebar/bottom nav
├── app.js                  ← router, όλες οι οθόνες/views
├── db.js                   ← IndexedDB wrapper
├── i18n.js                 ← μεταφράσεις EL/EN
├── utils.js                ← helpers (PDF, WhatsApp/Viber links, AI, photos)
├── manifest.webmanifest    ← PWA manifest
├── sw.js                   ← service worker (offline cache)
├── icon-192.svg, icon-512.svg
├── serve.py                ← τοπικός server
└── README.md
```

## Πού αποθηκεύονται τα δεδομένα

**Όλα** τα δεδομένα μένουν στον browser σου (IndexedDB). Δεν στέλνεται τίποτα σε server. Για backup χρησιμοποίησε Ρυθμίσεις → **Εξαγωγή backup** για JSON αρχείο.

## Επόμενα βήματα (roadmap)

- **Cloud sync** (Supabase/Firebase) για πολλαπλούς χρήστες
- **Multi-user / ρόλοι** (μηχανικός, ταμίας, ιδιοκτήτης)
- **Inventory ανταλλακτικών** με stock & κωδικούς ΕΑΝ
- **Φόρμα ραντεβού** με ημερολόγιο
- **Push notifications** για υπενθυμίσεις
- **QR code** ανά όχημα (γρήγορη πρόσβαση στο ιστορικό σκανάροντας)
- **Sign-in με Google** για cross-device συγχρονισμό
- **Ενσωμάτωση Twilio/Vonage** για αυτόματη αποστολή SMS/WhatsApp χωρίς να ανοίγει η εφαρμογή
- **Σύνδεση με συνεργαζόμενους προμηθευτές** ανταλλακτικών (API)
